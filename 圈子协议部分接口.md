# 圈子协议部分接口

## 1.获取所有精选

#### url: http://39.107.249.187:8080/jqkj/cricle/getSelectedCircle

#### 请求方式：GET

#### 请求参数：

| 属性 | 类型   | 说明   | 示例 |
| ---- | ------ | ------ | ---- |
| page  | int  | 第几页，不传默认为1           |1|
| limit | int  | 一页有多少条数据，不传默认为6 |6|

#### 返回参数：

| 属性   | 类型         | 描述                           |
| ------ | ------------ | ------------------------------ |
| data   | List<Object> | 返回的列表数据                 |
| msg    | String       | 返回的描述                     |
| code   | Int          | 返回的状态码：0 成功 ，-1 失败 |
| count  | int          | 总条数                         |
| result | String       | 返回的状态                     |

#### data：

| 属性         | 类型   | 描述                      |
| ------------ | ------ | ------------------------- |
| id           | int    | 发布的id                 |
| name  | String | 用户名称                  |
| type  | int    | 发布类型：1 加精 0 未加精 |
| create_time  | String | 发布时间              |
| title | String | 发布标题                |
| uid          | String | 用户id                    |
| video_path   | String | 视频地址                  |
| follow_num   | int    | 关注的数量                |
| give_num     | int    | 点赞的数量                |
| comment_num  | int    | 评论的数量                |
| transfer_num | int    | 转存的数量                |
| video_image | String | 预游览视频的截图        |
| video_len | String | 视频的时长 |
| headimgurl | String | 用户头像 |
| video_size | String | 视频的大小 |
| play_num | Int | 播放次数 |
| pubsum | int | 用户的资源数量 |
| isAdd | int | 当前用户是否加入了圈子 0 已加入 1 未加入2 待审核 |

## 7.获取一个发布视频的所有评论

#### **url:http://39.107.249.187:8080/jqkj/cricle/getCircleComment**

#### **请求方式：GET**

#### **请求参数：**

| 属性     | 类型 | 描述           | 示例 |
| -------- | ---- | -------------- | ---- |
| circleId | int  | 发布的视频的id | 26   |

#### **返回参数：**

| 属性   | 类型         | 描述                           |
| ------ | ------------ | ------------------------------ |
| data   | List<Object> | 返回的列表数据                 |
| msg    | String       | 返回的描述                     |
| code   | Int          | 返回的状态码：0 成功 ，-1 失败 |


#### data：

| 属性         | 类型   | 描述                      |
| ------------ | ------ | ------------------------- |
| id           | int    | id                    |
| messages | String | 评论的内容          |
| uid | String | 哪个用户id进行的评论 |
| circleId | Int | 发布视频的id |
| create_time | String | 评论的时间 |
| circleMine | Object | 用户(哪个用户进行的评论)的相关信息 |

#### **circleMine:**

| 属性        | 类型   | 描述                      |
| ----------- | ------ | ------------------------- |
| id          | int    | id                        |
| uid         | Stirng | 用户uid                   |
| circleName  | String | 用户的名字                |
| downLoad    | int    | 下载数量                  |
| resourceNum | int    | 资源数量                  |
| pirce       | int    | 所拥有的金币              |
| isCircle    | int    | 是否拥有圈子 0 有，1 没有 |
| num         | String | 圈子专属码                |
